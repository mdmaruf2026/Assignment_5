 1.What is the difference between var, let, and const?
 Ans:
    var: var is the old way for declaring variables. It can be re-declared and updated anywhere in the function.

    let: let is a modern way for declaring the variables.It can be updated but cannot be re-declared in the same block.

    const: const is a way which is used for fixed values. It cannot be updated or re-declared once assigned.

 2.What is the spread operator (...)?
 Ans:
    The spread operator is declared as (three dots ...). It is used to unpack or spread elements from an array or object. It is very useful for copying arrays or combining multiple arrays into one.

 3.What is the difference between map(), filter(), and forEach()?
 Ans:
   In JavaScript, map(), filter(), and forEach() are array methods used to perform operations on array elements. All of these purposes and outputs are different.

    map() : This method is used to create a new array by transforming each element of the original array. It applies a function to every element and returns the modified results in a new array.

    filter() : This method is used to select elements from an array based on a condition. It returns a new array containing only the elements that satisfy the given condition.

    forEach() : This method is used to iterate through each element of an array and perform an action, but it does not return a new array.

 4.What is an arrow function?
 Ans:
    An arrow function is a short way to write functions in JavaScript. It uses the => sign, which makes the code look cleaner and easier to read.
